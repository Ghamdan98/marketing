# Marketing E-Commerce

A modern e-commerce web application built with Laravel.

## 📌 About

This project is an online store developed using Laravel. It provides a simple shopping experience for customers and an administration panel for managing products, categories, and orders.

> **Note:** This project is still under development, and new features are being added regularly.

---

## 🚀 Features

### Customer

* User Registration & Login
* Browse Products
* Search Products
* Browse Products by Category
* Shopping Cart
* Checkout
* Order History
* User Profile Management

### Admin

* Dashboard
* Product Management (CRUD)
* Category Management (CRUD)
* Order Management
* Customer Management

## 📂 Installation

Clone the repository:

```bash
git clone https://github.com/Ghamdan98/marketing.git
```

Install dependencies:

```bash
composer install
npm install
```

Copy environment file:

```bash
cp .env.example .env
```

Generate application key:

```bash
php artisan key:generate
```

Run migrations:

```bash
php artisan migrate
```

Start the development server:

```bash
php artisan serve
```
